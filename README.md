# PenBridge 发布助手

思源笔记插件：一键将当前文档发布到**腾讯云开发者社区**。

## 功能

- **设置面板**：粘贴腾讯云社区 Cookie 并验证登录状态（`skey` 字段会被自动过滤，不会发送）
- **顶栏按钮**：对当前打开的文档快速发布
  - 编辑标题（≤80 字）
  - 搜索并选择标签（1~5 个）
  - 选择文章来源（原创 / 转载 / 翻译）
  - 自动创建/更新草稿后发布；已发布过的文档会自动更新原文章

## 使用方法

1. 浏览器登录 [cloud.tencent.com](https://cloud.tencent.com)，复制 Cookie（需包含 `qcommunity_session` 和 `uin`）
2. 打开插件设置，粘贴 Cookie，点击「验证并保存」
3. 打开要发布的文档，点击顶栏的发布按钮，填写信息后发布

## 注意事项

- 正文纯文本需不少于 140 字（社区平台限制）
- 文章发布后需平台审核（审核中 / 已发布 / 未通过）
- 本地图片（思源 assets）暂不支持自动上传
